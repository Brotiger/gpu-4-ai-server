# gpu-4-ai-server

HTTP API сервер, принимающий задачи от клиентов и балансирующий их между воркерами (узлами с GPU) по gRPC.

## Структура
- main.go — точка входа
- internal/ — служебные пакеты (балансировка, конфиги и т.д.)
- proto/ — gRPC-протокол (копия proto/worker.proto)

## Запуск
- `go run ./server`

## Требования
- Go 1.24+
